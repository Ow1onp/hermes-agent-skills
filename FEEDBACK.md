# 📬 Feedback

> Your feedback shapes hermes-agent-skills. Here's how to reach us.

---

## 🐛 Reporting a Bug

Found something broken? [Open a Bug Report](https://github.com/Ow1onp/hermes-agent-skills/issues/new?template=bug_report.yml).

**Before you submit:**
- Search [existing issues](https://github.com/Ow1onp/hermes-agent-skills/issues?q=is%3Aissue+label%3Abug) to avoid duplicates
- Include your environment (OS, Python version, Hermes Agent version)
- Provide clear reproduction steps
- Describe what you expected vs what actually happened

---

## 💡 Requesting a Feature

Have an idea to make things better? [Open a Feature Request](https://github.com/Ow1onp/hermes-agent-skills/issues/new?template=feature_request.yml).

**A strong feature request includes:**
- The problem you're trying to solve
- Your proposed solution
- Alternatives you've considered
- A concrete use case

---

## 🧬 Requesting a New Skill

Want a skill that doesn't exist yet? [Open a Skill Request](https://github.com/Ow1onp/hermes-agent-skills/issues/new?template=skill_request.yml).

**A strong skill request includes:**
- The recurring problem this skill would solve
- An example workflow (step by step)
- The expected outcome (what "done" looks like)

---

## 📖 Reporting a Documentation Issue

Found unclear, missing, or outdated docs? [Open a Documentation Issue](https://github.com/Ow1onp/hermes-agent-skills/issues/new?template=documentation_issue.yml).

**Include:**
- The specific page or file path
- What's wrong (unclear / missing / outdated / wrong)
- Your suggested fix

## 🤝 Contributing

Want to go beyond feedback? We welcome contributions.

**Read the full guide:** [CONTRIBUTING.md](CONTRIBUTING.md)

**Quick start:**

```bash
# 1. Install
pip install hermes-agent-skills

# 2. Scaffold a new skill
hermes-skill create my-skill-name

# 3. Edit the generated SKILL.md

# 4. Validate
hermes-skill validate skills/<phase>/my-skill-name/SKILL.md

# 5. Run tests
pytest tests/ -v

# 6. Open a Pull Request
```

**What we're looking for:**
- New skills that solve recurring problems
- Improvements to existing skills
- Bug fixes and documentation updates
- Creative uses of Hermes-specific features (`delegate_task`, `cronjob`, etc.)

**First contribution?** Look for issues labeled [`good-first-issue`](https://github.com/Ow1onp/hermes-agent-skills/issues?q=is%3Aissue+label%3Agood-first-issue+).

---

## 📊 What Happens to Your Feedback?

1. **All submissions are triaged** — labeled by type and priority
2. **Bugs (P0)** get fixed first — usually within days
3. **Features** are weighed against the roadmap and community demand
4. **Skill requests** are tracked and may become `good-first-issue` candidates

Your feedback directly shapes the priorities for each release. See the [CHANGELOG](CHANGELOG.md) for what's shipped so far.
